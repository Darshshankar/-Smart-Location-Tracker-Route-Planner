📍 Smart Location Tracker & Route Planner

A mobile application built using React Native that allows users to plan routes, track locations, and calculate distance between two points in real-time using Google Maps and Firebase.

---

🚀 Features

- 📍 Real-time location tracking
- 🗺️ Route planning with map visualization
- 📏 Distance calculation using Haversine formula
- 🔄 Start & destination selection (manual + map)
- 📷 QR code scanning for quick input
- ⚡ Smooth UI with navigation flow

---

🛠 Tech Stack

- React Native
- Firebase Realtime Database
- Google Maps API
- JavaScript
